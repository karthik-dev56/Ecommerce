## Installation & Setup

1. **Clone the repository:**
   ```sh
   git clone https://github.com/your-username/ecommerce-react.git
   cd ecommerce-react
   ```

2. **Install dependencies:**
   ```sh
   npm install
   ```

3. **Start the development server:**
   ```sh
   npm start
   ```

4. **Build for production:**
   ```sh
   npm run build
   ```
